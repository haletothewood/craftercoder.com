# CrafterCoder.xyz

To get started:
```sh
gatsby develop
```

To build:
```sh
gatsby build
```

To serve:
Any static server will do, [ngrok](https://ngrok.com/), [netlify](https://www.netlify.com/), [heroku](https://dashboard.heroku.com/) whatver! I use netlify, deployed via [circleCI](https://circleci.com/)
